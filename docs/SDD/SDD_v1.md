# Documento de Diseño de Software (SDD) - v1.0

## 1. Arquitectura
Se utiliza una arquitectura de capas simple para el módulo de autenticación de LoginCRM.

## 2. Componentes Técnicos
* **Frontend:** Formulario simple en HTML/JS.
* **Lógica:** Función de validación `validarAcceso` en JavaScript.
* **Seguridad:** Manejo de constantes para credenciales de prueba.

## 3. Decisiones Técnicas
Se optó por JavaScript puro para garantizar la reproducibilidad inmediata de la línea base v1.0 sin dependencias externas.
