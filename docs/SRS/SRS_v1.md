# Especificación de Requisitos de Software - v1.0

## Requisitos Funcionales
1. **RF-01:** El sistema permitirá el acceso mediante correo y contraseña.
2. **RF-02:** Validación de formato de email corporativo.
3. **RF-03:** Redirección automática al panel principal tras login exitoso.
4. **RF-04:** Mensaje de error descriptivo ante credenciales inválidas.

## Requisitos No Funcionales
5. **RNF-01:** La validación de credenciales debe responder en menos de 2 segundos.
6. **RNF-02:** Interfaz compatible con navegadores modernos (Chrome, Edge).

---
## Control de Cambios Post-Baseline
* **REQ-007:** Bloqueo de cuenta tras 3 intentos fallidos de inicio de sesión.
